# Security Policy

## Reporting Vulnerabilities

If you discover a security vulnerability in the NeuronGate API, please report it responsibly.

**Email:** security@neurongate.net

Please include:
- Description of the vulnerability
- Steps to reproduce
- Potential impact

**Do NOT:**
- Open a public issue for security vulnerabilities
- Exploit the vulnerability beyond proof-of-concept
- Access other users' data

## Response Timeline

- **Acknowledgment:** Within 24 hours
- **Initial assessment:** Within 72 hours
- **Fix deployed:** Depends on severity (critical: <24h, high: <1 week)

## Scope

In scope:
- neurongate.net API endpoints
- Authentication and authorization bypasses
- Data exposure vulnerabilities
- Rate limit bypasses

Out of scope:
- Social engineering attacks
- Denial of service (volumetric)
- Third-party dependencies with known CVEs (report separately)

## Hall of Fame

We credit responsible reporters (with permission) in our security acknowledgments.
