# Changelog

All notable changes to the NeuronGate API will be documented here.

## [2026.4] — 2026-04-14

### Added
- Billing dashboard with printable invoices
- Invoice detail pages with downloadable HTML
- Usage analytics with per-model breakdown
- Multi-key management with scoped permissions
- Automatic failover between providers

### Changed
- Improved streaming response latency
- Updated model pricing to reflect latest provider changes

## [2026.3] — 2026-03-15

### Added
- Initial public launch
- 50+ models from 8 providers
- OpenAI-compatible /v1/chat/completions endpoint
- Crypto payments (USDT, USDC, ETH, BTC)
- Real-time usage tracking
- API key management

### Security
- TLS 1.2/1.3 enforced on all endpoints
- Rate limiting with configurable tiers
- No prompt/completion data retention
