# Contributing

Thanks for your interest in contributing to NeuronGate's documentation and examples!

## How to Contribute

### Reporting Issues
- Found a bug in the docs? Open an issue.
- API behaving differently than documented? Let us know.

### Submitting Changes
1. Fork the repo
2. Create a branch (`git checkout -b fix/typo-in-auth-docs`)
3. Make your changes
4. Submit a pull request

### What We Accept
- Documentation fixes (typos, clarity improvements)
- New code examples
- Translation contributions
- Bug reports

### What We Don't Accept
- Changes to the API specification (that's internal)
- Promotional content or spam

## Code Examples Guidelines

When contributing code examples:
- Use the OpenAI SDK (Python or JavaScript)
- Base URL: `https://neurongate.net/v1`
- Use placeholder API keys: `ng-your-api-key`
- Include error handling
- Add comments explaining non-obvious steps
- Test your code before submitting
